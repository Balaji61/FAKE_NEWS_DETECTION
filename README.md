# Fake News Detection Using Machine Learning

This project uses TF-IDF Vectorization and Logistic Regression to classify news articles as **Fake** or **Real**.

### Features
- Python + Scikit-learn ML model
- TF-IDF feature extraction
- Logistic Regression classifier
- Flask web interface
- Trained on fake_news.csv dataset

### How to Run
python app.py
